Terminalo komandos

  • ls / dir - parodo aplankus ir failus dabarnineje direktorijoje
  • cd - changeDirectory - keičia direktorija
  • TAB - skirtas užpildyti terminalo komandą arba pateikti pasiūlymus
  • CTRL C - atšaukia vykdomą veiksmą / nutraukia komandą
  • vidurinis / dešinys pelės mygtukas - įklijuoja tekstą į komandinę eilutę
  • ARROW UP/DOWN - vaikščiojimas per terminalo kodų istoriją

Git Komandos

  • git init - inicializuoja git repozitoriją
  • git status - parodo git repozitorijos statusa
  • git add ./*/file - pridėti failus prie paruoštų commit'inimui failų
  • git commit -m "žinute" - sukuriama kodo versija su žinute
  • git restore ./*/file - panaikina visus šios kodo versijos pakeitimus (kurie nėra add'inti)
  • git restore --staged ./*/file - panaikina add'intus pakeitimus
  • git push - siunčia naujas kodo versijas į github
  • git log - parodo kodo versijų istoriją
  • git checkout commitID - nukeliauja i kodo versijų istoriją
