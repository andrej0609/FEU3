Tag'ų sarašas

  • p - paragraph - Paragrafas, skirtas rašyti pastraipai
  • h1-h6 - Heading1-Heading6 - Antraštės, skirtos rašyti pavadinimams ir tarpusavyje skiriasi teksto dydžiu
  • img - Image - Paveiksiukas, naudojamas katru su src ir alt atributais
  • a - Hypertext Anchor (link) - Nuoroda
  • hr - Thematic Break (horizontal line) - Sukuria eilutės lūžį su linija
  • br - Line Break - Eilutės lūžis
  • ul - Unordered List - Nerikiotas sarašas
  • ol - Ordered List - Rikiotas sarašas
  • li - List Item - Sarašo elementas (skirtas rikiotam arba nerikiotam sarašui)
  • meta - Skirtas nurodyti puslapio meta duomenis
  • title - Skirtas nurodyti puslapio tab'o pavadinimą

Atributu sarašas

  • lang - language - Nurodo kokia kalba vyrauja tame tag'e (naudinga automatiniams skaitytuvams)
  • charset - Character set - Simbolių rinkinys nurodo kokią simbolių biblioteką naudoti
  • src - Source - Šaltinis nurodantis failą
  • alt - Alternative text - Alternatyvus tekstas paveiksiuko neužkrovimo atveju
  • width - Nurodo elemento plotį
  • height - Nurodo elemento aukštį
  • href - Hyper link - Nuoroda į puslapį arba failą
  • target - Nurodo kur atidaryti nuoroda

Klaviatūros trumpinių sarašas

  • CTRL S - Išsaugoja faila
  • CTRL C - Kopijoja pažymeta informacija
  • CTRL X - Iškerpa pažymeta informacija
  • CTRL Z - Atšaukia paskutinį veiksmą/us
  • CTRL V - Iklijuoja kopijuota/iškirpta informacija
  • CTRL F - Atidaro greita paieška
  • CTRL A - Pažymėti visus elementus dokumente arba lange
  • CTRL F4 - Uždaryti aktyvųjį elementą arba išeiti iš aktyviosios programos
  • Ctrl Home - Eiti į dokumento viršų
  • Ctrl End - Eiti į dokumento apačią
  • Tab - Tabulioja pažymeta koda i dešine
  • SHIFT TAB - tabulioja pažymeta koda i kaire
  • CTRL / - Komentuoja pažymetas eilutes
  • CTRL Y - Atšaukia CTRL Z veiksma/us
  • Alt Z - Word Wrap
  • CTRL R - Duoda pasirinkti ka atidaryti iš failu/aplanku istorijos
  • CTRL SHIFT L - Pažymi visus vienodus žodžius/sakinius, taip pat galima juos keisti
  • CTRL SHIFT K - Ištrina pažymeta eilutę
  • ctrl Shift I/J F12- Atidaro Developer Tools langą naršyklėje
  • CTRL D - Pažymi toki pati žodi einanti toliau kode
  • SHIFT ALT UP/DOWN - Dublikuoja pažymeta teksta i viršu i aukštyn/žemyn
  • ! - Sukuria HTML failo karkasa
  • CTRL ENTER sukuria nauja eilute (nelaužant kitos)
  • CTRL SHIFT C - Naršykleje parenka inspect element iranki
  • ALT TAB - Vaikščioti WINDOWS atidarytose languose
  • WINDOWS TAB - Atidaro desktop'u pasirinkimo lango
  • CTRL arrowRIGHT/LEFT - Vaikščioti per visą žodi
  • SHIFT vaikščiojimo per kodą kombinacija - Pažymi žodžius per kurios kursorius juda
  • CTRL T - Atidaro nauja tab'a (naršyklėje)
  • CTRL SHIFT T - Atidaro paskutini/us uždarytus tab'a/us
