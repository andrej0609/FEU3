CSS rašymo būdai

Inline - Linijinis

Linijinis stilius yra, kai elementui, kurį stilizuojame suteikiame style atributą ir jo viduje aprašome stilius. Neturi perpanaudojimo galimybės ir yra skirtas tiktais testavimui.

Internal - Vidinis

Vidinis stilius yra, kai HTML failo viduje, head tag'o dalyje pasirašome style tag'ą ir tame style tag'e rašome CSS kodą. Turi šiek tiek perpanaudojimo galimybių (nes turi selektorius) ir dažniausiai naudojamas testavimui.

External - Išorinis

Išorinis stilius yra rašomas atskirame CSS faile ir yra prijungiamas prie HTML failo naudojant link tag'ą. Turi visas reikalingas/norimas perpanaudojimo galimybes, yra plačiausiai naudojamas ir galutinio projekto variante, turėtų likti tik šis stiliaus rašymo būdas.

Basic Selectors

Element / Tag

Selektinami visi tag'ai pagal nurodytą elementą.
Pvz.: h2 -> parinks visus h2 elementus esančius HTML'e.

Class

Klasė sukuriame kokiu tik norime logišku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jog kuriame klasę - sugalvoto pavadinimo pradžioje parašome tašką. HTML'e norint priskirti klasę elementui, tam elementui suteikiame atributą "class" ir jam priskiriame sugalvotą klasės pavadinimą be taško.
Pvz.: CSS'e: .klasesVardas; HTML'e: <p class="klasesVardas">
Klasės turi daug su daug santykį. Vienas elementas gali turėti daugelį klasių ir viena klasė gali būti priskirta daugeliui elementų.

ID

ID sukuriame kokiu tik norime logišku ir trumpi pavadinimu. CSS'e tam, kad nurodytume, jog kuriame ID - sugalvoto pavadinimo pradžioje parašome groteles. HTML'e norint sriskirti ID elementui, tam elementui suteikiame atributą "id" ir jam prisikiriame sugalvotą ID pavadinimą be grotelių.
Pvz.: CSS'e: #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi vienas su vienu santykį. Vienas elementas gali turėti TIK vieną ID. Ir vienas ID turėtų būti priskirtas TIK vienam elementui.
