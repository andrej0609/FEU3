CSS rašymo būdai

Inline - Linijinis

Linijinis stilius yra, kai elementui, kuri stilizuojame suteikiame style atributa ir jo viduje aprašome stilius.
Neturi perpanaudojimo galimybes ir yra skirtai tiktai testavimui.

Internal - Vidinis

Vidinis stilius yra, kai HTML failo viduje, head tag'o dalyje pasirašome style tag'ą ir tame style tag'e rašome CSS kodą.
Turi šiek tiek perpanaudojimo galimybių (nes turi selektorius) ir dažniausiai naudojamas testavimui.

External - Išorinis

Išorinis stylius yra rašomas atskiriame CSS faile ir yra prijungiamas prie HTML failo naudojant link tag'a.
Turi visas reikalingas/norimas perpanaudojimo galimybes, yra plačiausiai naudojamas ir galutinio projekto variante turetu likti tik šis stiliaus rašymo budas.

Basic selectors -

Element / Tag

Selektinami visi tag'ai pagal nurodita elementa.
Pvz.: h2 -> parinks visus h2 elementus esančius HTML'e.

Class

Klasė sukuriama kokiu tik norime logišku ir trumpu pavadinimu. CSS'r tam, kad nuroditume, kad kuriame klasė, sugalvoto pavadinimo pradžioje parašome taška.
HTML'e norint priskirti klasė elementui, tam kad elementui suteikiama atributą "class" ir jam priskiriame sugalvotą klasės pavadinimą be taško.
Pvz.: CSS'e: .klasesVardas; HTML'e: class="klasesVardas"; <p class="klasesVardas">
Klases turi daug su daug santykii. Vienas elementas gali tureti daugeli klasiu ir viena klase gali buti priskirta daugeliui elementų.

ID

ID sukuriame kokiu tik norime logišku ir trumpu pavadinimu. CSS'e tam, kad norodytume, jog kuriame ID - sugalvoto pavadinimo pradžioje parašome groteles. HTML'e norint priskirti ID elementui, tam elementui suteikiame atribut'a "id" ir jam priskiriame sugalvota ID pavadinimą be groteliu.
Pvz.: CSS'e: #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi vienas su vienu santikį. Vienas elementas gali tureti TIK viena ID. Ir vienas ID turetų būti priskirtas TIK vienam elementui.

Colors

Names

Spalvos gali turėti vardus. Vardinių spalvų yra 140. Norint priskirti elementui spalvą, priskiri tos spalvos vardą.
Pvz.: red; blue; green; lightgrey; darkgreen; ...

RGB

RGB - Red Green Blue. Kiekvienos iš šių triju spalvou kiekio galima ideti nuo 0 iki 255 ir šitaip galimaišgaut bet kurią spalvą.
Pvz.: rgb(0, 0, 0) - juoda. rgb(255, 0, 0) - rauduona; ...
RGBA - Red Green Blue Alpha. Alpha nurodo spalvos permatomumą ir jis gali būti nuo 0 iki 1 tūkstantają paklaida.
Pvz.: rgba(0, 255, 0, 0) - Nematoma (žalia);
rgba(0, 255, 0, 1) - pilnai matoma (žalia); ...

Hex

Hex - Hexadecimal - Šešioliktainis. Kiekvienos oš spalvos (RGB) galima nuroditi nuo 00 iki ff ir šitaip išgauti bet kurią spalvą.
Skaičiavimas šešioliktainėje sistemoje: 00 01 02 03 04 05 06 07 08 90 0a 0b 0c 0b 0e 0f 10 11 12 13 ..... 19 1a 1b ... 1f 20 21 22 ... 9f a0 a1 a2 ... a9 aa ab ac ae af b0 .... ff
Pvz.: #000000 - juoda; #ff0000 - rauduona; ...
Trumpiniai: #000 - juoda; #f00 - rauduona; ...
Spalvos matomumas su Hex: gale prisideda dar du ŠeŠioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas.
Pvz.: #00ff0000 - nematoma (žalia); #00ff00ff - pilnai matoma (žalia); ...
Trumpiniai: #0f00 - nematoma (žalia); #0f0f - pilnai matoma žalia; ...

Positions

Static

Statinė pozicija nieko nedaro. Nei pakeičia elemento paziciją, nei suteikia jam galimybę naudotis top, left, right, bottom, z-index stiliais.
Statinė pozicija gali būti suteikiama kaip placeholder'is, arba panaudota tam, kad nuimti kokią kitą poziciją.

Absolute

Elementas su absolute pozicija išeina iš puslapio tekmes. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais. Fixed
