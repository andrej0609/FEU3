CSS Framework - kažkieno kito parašytas CSS kodas, kuri tu gali naudoti.
Norint nauduoti kažkieno framework'a,reikia ji prijungti prie savo kodo (CSN).
Dauguma framework'u veikia suteikiant ju aprašytas klases kažkokiems HTML elementams.
Yra Classless CSS framework'u. Juos naudojant, nereikia suteikti klasiu jokiems HTML elementams. Viskas aprašyta naudojant tag'u selektorius.
