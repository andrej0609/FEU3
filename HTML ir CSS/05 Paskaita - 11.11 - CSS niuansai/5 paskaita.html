Vaikinis / Tevinis elementas

Tevinis elementas gali tureti daugeli vaikiniu elementu.
Vaikinis elementas, gali tureti tik viena tevini elementa.

Paveldejimas

Paveldeti stilius, gali tik vaikinis elementas iš tevinio elemento.
Paveldimi stiliai yra tie, kurie keičia elemento išvaizda, ne būseną. Pvz.: color, font-size, font-family....
Nepaveldimi stiliai yra tie, kurie keičia elemento būseną. Pvz.: width, height, display, border, position...
Net paveldeti stiliai, gali nebuti atvaizdioti del stiliu prioritetu.

Numatytieji styliai

Numatytieji stiliai yra tag'ams automatiškai naršyklės suteikti stiliai. Tarp skirtingu naršykliu, numatytieji stiliai gali minimalistiškai keistis ir šitaip sugadinti svetaines vaizda skirtingu naršykliu vartotojams.
Tam, kad nebutu problemu kuriant dizaina skirtingoms naršyklems, yra naudojami Reset arba Normalize CSS'ai.
Reset - resetina visus stilius (ištrina stilius).
Normaliz - uvienodina visus stilius tarp naršykliu ir panaikina dali kenkejišku stiliu.

  2. ID - 100
  3. Class - 10
  4. Element - 1

  8. Inheritance
Google

Esant dviem selektoriam, kurie nusako to paties elemento vienodus stilius, yra atsižvelgiama i tu selektoriu prioritetus. Aukštesni bala turintys prioritetai, turi aukštesni prioriteta.
Esant dviem vienodo prioritetu balo selektoriams, keičiant ta pati elemento stiliu, yra atsižvelgiama i CSS eiliškuma. Kuris CSS stilius aprašytas veliau, tas ir bus pritaikomas.

Jeigu imanoma - nenaudoti!!!
!important prierašas prie stiliaus ignoroja visus prioritetus ir suteikia ta stiliu ir niekas jo negali pakeisti, apart kito !important prierašo, kuris turi aukštesni prioriteta.
