Kartojimas

  1. Kintamieji (what, why, scope, type, assign)
    • string - Žodinis kintamasis rašomas kabutese '' / "" / ``
    • number - Skaitinis kintamasis rašomas be kabičiu
    • boolean - Boolean tipas turi tik dvi reikšmes: true (tiesa) ir false(netiesa).
    • array - Masyvas pvz: let book = ['reiksme', 'reiksme', 'reiksme'...]
    • object - Objektas - pvz: let book = {raktas: 'reiksme' ; raktas1: 'reiksme1'; ......}
  2. Operatoriai
    • sudetys, atimtis, daugyba, dalyba
    • laipsnio kelimas , šaknies traukymas, liekanos radimas
    • lygu, lyyygu, nelygu, nelyyygu
    • daugiau, mažiau, daugiau arba lygu, mažiau arba lygu
    • IR, ARBA, NE
    • tipas (typeof) - Operatorius typeof grąžina argumento tipą. Jis naudingas kai mes norime išskirtinai apdoroti skirtingų tipų vertes arba norime greitai patikrinti tipą.
  3. Salygos
    • If else....else
    • switch case
    • truthy/falsy
    • ternary
    • AND
  4. Ciklai - yra kartojami nurodyti veiksmai, tol kol yra tenkinama salyga.
    • for - for naudojamas tuomet, kai gali apskaicioti/zinuoti kiek sykiu kartosis ciklas.
    • while - while ciklas naudojamas tuomet, kai negali apskaicioti/zinoti kiek kartu reikes vykdyti cikla.
    • do while - Bent viena syki butu paleistas ciklas.
    • for in - suka cikla per objekta.
    • for of - - suka ciklas per masyva.
    • array iteration methods
  5. Funkcijos
    • funkcijos deklaravimas
    • parametrai / argumentai
    • funkcijos gražinimas (return) - funkcija gali kazka grazinti. Funkcijos veiksmai vyksta iki tol kol pasiekiamas return'as.
    • lambda
  6. Methods
    • Math
    • string
    • array
