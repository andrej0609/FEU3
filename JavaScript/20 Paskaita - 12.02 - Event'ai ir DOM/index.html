Events

Ivykiai - puslapyje vykstantys kažkokie veiksmai/ivykiai (peles paspaudimas, klaviaturos paspaudimas, informacijos pažymejimas, scroll'inimas, formos pateikimas, failo ikelimas/atsiuntimas...)
Noredami klausyti ar nutinka koks ivykis, turime pradžioje nuroditi (selektinti) kurioje DOM dalyje tas ivykis gali nutikti.
Nusirodžius (selektinus), kurioje dalyje klausomes ivykio - reikia aprašyti ivykio klausymasi. Tai padarome rašydami "addEventListener" metoda, kuris priima 2 parametrus: koksIvykis, kaDarytiNutikusIvykiui.

Document Object Model

Puslapis - kuriame yra visa vartotojui atvaizduojama informacija(tekstas, paveiksliukas, forma, lentele......).
Objektas - kažkoks puslapio elementas (nuo HTML tag'o, iki ikonos ar span tag'o).
Modelis - objektu struktura dokumente ( tekstas, paveiksliukas, forma, lentele...).

Noredami per JS selektinti kažkuria DOM dali, turime rašyti toki koda: "document.querySelector" arba "document.querySelectorAll". "querySelector" priima 1 parametra - selektoriu (toki koki rašeme CSS'e). (ಥ﹏ಥ)
